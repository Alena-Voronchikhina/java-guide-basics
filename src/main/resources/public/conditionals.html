Conditionals — Making Decisions

Start Variables Conditionals Loops Arrays 2D Arrays Important Rules

With Hardcoded Values

int score = 85; if (score >= 90) { System.out.println("A"); } else if (score >= 80) { System.out.println("B"); } else if (score >= 70) { System.out.println("C"); } else { System.out.println("F"); }

With User Input

System.out.print("Enter your score: "); int score = sc.nextInt(); if (score >= 90) { System.out.println("A"); } else if (score >= 80) { System.out.println("B"); } else if (score >= 70) { System.out.println("C"); } else { System.out.println("F"); }

Other Conditional Forms

String result = (score >= 60) ? "Pass" : "Fail"; System.out.print("Enter day: "); String day = sc.nextLine(); switch (day) { case "Monday": System.out.println("Start of week"); break; case "Friday": System.out.println("Almost weekend"); break; default: System.out.println("Regular day"); }
String comparison: NEVER use == for Strings. Always use .equals(). if (name == "Alena") // wrong if (name.equals("Alena")) // correct
