What Intro Java Can Do with 2D Arrays

  • Store rows and columns of values (like a table or grid).
  • Loop through all cells to compute totals and averages.
  • Useful for grade books, seats, and small game boards.

Hardcoded 2D Array

int[][] grades = { {90, 85, 88}, {76, 92, 81}, {100, 95, 98} }; System.out.println(grades[0][1]); // row 0, col 1 => 85 System.out.println(grades[2][2]); // row 2, col 2 => 98

User Input into 2D Array

import java.util.Scanner; Scanner sc = new Scanner(System.in); int[][] table = new int[2][3]; for (int row = 0; row < table.length; row++) { for (int col = 0; col < table[row].length; col++) { System.out.print("Enter value for [" + row + "][" + col + "]: "); table[row][col] = sc.nextInt(); } }

Process All Values (Sum + Average)

int sum = 0; int count = 0; for (int row = 0; row < table.length; row++) { for (int col = 0; col < table[row].length; col++) { sum += table[row][col]; count++; } } double average = (double) sum / count; System.out.println("Sum = " + sum); System.out.println("Average = " + average);

Common Mistakes
