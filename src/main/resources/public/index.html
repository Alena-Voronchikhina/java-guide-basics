How to read this guide:

📌 Blue code boxes = hardcoded values (you set them in code).
📌 Yellow code boxes = user provides values (via Scanner).
📌 Every program you write in class will use one or the other. Learn both.

The Skeleton — Every Program Starts Here

public class MyProgram { public static void main(String[] args) { // your code goes here } }
  • File name MUST match class name: MyProgram.java
  • Every statement ends with ;
  • Java is case-sensitive: String is not string

Getting User Input — The Setup

import java.util.Scanner; public class MyProgram { public static void main(String[] args) { Scanner sc = new Scanner(System.in); } }

How to Read Different Types

int age = sc.nextInt(); double price = sc.nextDouble(); String word = sc.next(); String line = sc.nextLine();
THE SCANNER TRAP: After nextInt() or nextDouble(), Enter stays in buffer. If you call nextLine() right after, you may get empty text. int num = sc.nextInt(); sc.nextLine(); // eat leftover Enter String name = sc.nextLine();
