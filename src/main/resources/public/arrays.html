What Arrays Let Java Do

  • Store many values of the same type in one variable.
  • Loop through values to calculate totals, average, max, and min.
  • Update values by index and process data in order.

Hardcoded Array

int[] scores = {88, 92, 76, 95, 84}; System.out.println(scores[0]); // first item System.out.println(scores[4]); // last item

Array with User Input

import java.util.Scanner; Scanner sc = new Scanner(System.in); int[] values = new int[5]; for (int i = 0; i < values.length; i++) { System.out.print("Enter value " + (i + 1) + ": "); values[i] = sc.nextInt(); }

Process an Array (Sum and Average)

int sum = 0; for (int i = 0; i < values.length; i++) { sum += values[i]; } double avg = (double) sum / values.length; System.out.println("Sum = " + sum); System.out.println("Average = " + avg);

Common Mistakes

// Index out of bounds (wrong) int x = values[values.length]; // Correct last index int last = values[values.length - 1]; // Arrays keep fixed size after creation
